What is an agentic workflow? When the Fraud Detection Agent scores a message above a risk threshold, it autonomously hands off to the Identity Verification Agent and Customer Service Agent — no human trigger required. This page captures each such chain as a real-time trace with per-agent timestamps, confidence scores, and the final outcome.
